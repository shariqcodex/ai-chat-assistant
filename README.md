# AI Chat Assistant

A conversational AI chatbot built with Python and Flask, powered by Llama 3.3 70B.

## Features
- Real-time AI responses
- Persistent chat history (survives restarts)
- Clean browser-based UI
- Secure API key management

## Tech Stack
- Python, Flask, Groq API, HTML/CSS/JavaScript

## How to Run
1. Clone the repo
2. Add your Groq API key to a `.env` file
3. Run `pip install flask groq python-dotenv`
4. Run `python app.py`
5. Open `http://127.0.0.1:5000`
